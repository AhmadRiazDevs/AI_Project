# AI_Project

A Python-based traffic simulation project that models vehicle movement and traffic signal behavior at an intersection. The simulation is built with Pygame and visualizes how vehicles move, stop, and interact with changing traffic signals.

## Features

- Interactive traffic intersection simulation
- Vehicle generation from multiple directions
- Traffic signal logic with green, yellow, and red states
- Visual vehicle movement and signal timing
- Simple terminal-based traffic status output

## Project Structure

- `simulation.py` - Main simulation script
- `simulation Dy.py` - Alternate simulation version
- `images/` - Vehicle and signal images used by the simulation
- `first.png` - Background image for the intersection
